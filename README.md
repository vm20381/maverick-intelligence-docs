# Maverick Intelligence API Docs

Documentation for the [Maverick Intelligence](https://maverickintelligence.co)
Developer API — programmatic access to your identified website visitors, companies,
behavioral events, and analytics.

- **Live docs:** https://docs.maverickintelligence.co
- **API base URL:** `https://api-v1.maverickintelligence.co`
- **Get an API key:** https://app.maverickintelligence.co/api-keys

## Contents

- `openapi.yaml` — OpenAPI 3.0 spec for all endpoints
- `*.mdx` — Mintlify documentation pages (getting started, auth, rate limits, errors, examples)
- `docs.json` — Mintlify site configuration
